# AreaChart

The `AreaChart` component is a responsive chart built using the ReCharts library. It can be configured with various properties such as width, height, data key, stroke color, and fill color.

## Usage

<Canvas>
  <Story name="Default">
    <AreaChartStories.Default />
  </Story>
</Canvas>

## Available props

<Markdown>
  {`
| Name      | Type     | Default     | Description                        |
| --------- | -------- | ----------- | ---------------------------------- |
| width   | string | '100%'    | The width of the chart container.  |
| height  | number | 400       | The height of the chart container. |
| dataKey | string | 'uv'      | The key of the data to be plotted. |
| stroke  | string | '#8884d8' | The stroke color of the area line. |
| fill    | string | '#8884d8' | The fill color of the area.        |
| data    | object | null | The chart data.        |
`}
</Markdown>

## Best Practices

### Do

- **Use ResponsiveContainer:** Ensure the chart is wrapped in a **'ResponsiveContainer'** to make it responsive.
- **Configure data keys:** Use appropriate data keys to match the data structure you have.
- **Set appropriate dimensions:** Adjust the width and height to fit the layout of your application.

### Don't

- **Avoid hardcoding dimensions:** Do not hardcode the dimensions, use responsive properties where possible.
- **Ignore data structure:** Ensure the data structure matches the dataKey provided.

## Example

### Default AreaChart

```jsx
import ACNAreaChart from "./AreaChart";

<ACNAreaChart
  width="100%"
  height={400}
  dataKey="uv"
  stroke="#8884d8"
  fill="#8884d8"
  data={data}
/>;
```
